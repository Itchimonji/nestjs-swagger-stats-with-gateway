## Installation

```
$ npm install
$ nx serve 
// open url http://localhost:3333/api/ for swagger API
// open url http://localhost:3333/swagger-stats/ for swagger stats
// prometheus metrics: http://localhost:3333/swagger-stats/metrics for swagger stats
```
